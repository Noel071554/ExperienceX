# About ExperienceX
ExperienceX is a plugin! Have 8 commands.

# Requirements
EssentialsX currently supports CraftBukkit, Spigot. Other server implementations may work, but we don't recommend them as they may cause compatibility issues.

# Commands
/mobstick
/troll
/liferod
/playerinfo
/exgm <1, 2, 3, 4>
